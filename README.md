# 💼 Portfolio

Personal portfolio showcasing my journey as a full-stack developer. Built with Next.js, TypeScript and Tailwind CSS.

**Live:** [bzt.vercel.app](https://bzt.vercel.app)

---

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** React Icons (Feather Icons)
- **Animations:** CSS Transitions + Intersection Observer API
- **Deployment:** Vercel

---

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/vbzt/portfolio.git
cd portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
npm start
```
